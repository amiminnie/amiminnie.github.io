Design

Google Fonts & Icons

A multilingual library of fonts, anything you can think of, for completely free... That's the best thing a designer can dream for. Thankfully, Google has come forward with Google fonts. Not only that, but I have used their icon library for some of the icons used here!

Development

Github Pages

This whole project wouldn't have been possible without the awesome hosting service that is Github Pages. Finally being able to host my thoughts into the world without having to deal with hosting expenses or the hassle of registering a top level domain was a huge relief.

Supabase

When I was researching on how I can make a guestbook system, without having my own server, I came across Supabase. Its free tier is perfect for personal projects like mine, thus it now powers the whole article system.

Phoenix Code

While there were many options for a code editor when I first started web development, I went with Phoenix Code due to its simplicity, being both lightweight and open-source.
